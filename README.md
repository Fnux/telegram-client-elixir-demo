# TelegramClient

A simple example for [telegram_mt](https://github.com/Fnux/telegram-mt-elixir).

## Installation

```
git clone https://github.com/Fnux/telegram-client-elixir-demo
cd telegram-client-elixir-demo
mix deps.get
```
